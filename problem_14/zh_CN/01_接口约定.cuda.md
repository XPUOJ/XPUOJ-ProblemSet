你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_bf16.h>
#include <cuda_fp8.h>

extern "C" void run_kernel(
    const __nv_fp8_e4m3* a_data,
    const float* a_sf,
    const __nv_fp8_e4m3* b_data,
    const float* b_sf,
    const int32_t* masked_m,
    __nv_bfloat16* d,
    int64_t num_groups,
    int64_t max_m,
    int64_t n,
    int64_t k
);
```

### 参数说明

* `a_data`：输入指针（FP8E4M3），shape $(num\_groups, max\_m, k)$，行优先连续存储，只读
* `a_sf`：输入指针（float32），shape $(num\_groups, max\_m, \lceil K/128 \rceil)$，per-token 缩放因子，只读
* `b_data`：输入指针（FP8E4M3），shape $(num\_groups, n, k)$，每个组的 FP8 权重矩阵，只读
* `b_sf`：输入指针（float32），shape $(num\_groups, \lceil N/128 \rceil, \lceil K/128 \rceil)$，per-block 缩放因子，只读
* `masked_m`：输入指针（int32），shape $(num\_groups,)$，每个组的实际处理行数，只读
* `d`：输出指针（bfloat16），shape $(num\_groups, max\_m, n)$，行优先连续存储，需写入计算结果
* `num_groups`：分组数量（int64）
* `max_m`：每个组的最大行数（int64）
* `n`：b 的第二维、d 的第三维大小（int64）
* `k`：a 和 b 的最后一维大小（int64）

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 CUDA `__global__` kernel，完成 FP8 反量化和带掩码的分组矩阵乘法
