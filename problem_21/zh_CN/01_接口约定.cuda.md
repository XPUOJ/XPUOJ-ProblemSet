你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_fp8.h>

extern "C" void run_kernel(
    const __nv_fp8_e4m3* q,
    const __nv_fp8_e4m3* kv,
    const float* kv_sf,
    const float* weights,
    const int32_t* cu_seqlen_ks,
    const int32_t* cu_seqlen_ke,
    float* logits,
    int64_t M,
    int64_t seq_len_kv,
    int64_t num_heads,
    int64_t head_dim
);
```

### 参数说明

* `q`：输入指针（FP8 E4M3），shape `(M, num_heads, head_dim)`，只读
* `kv`：输入指针（FP8 E4M3），shape `(seq_len_kv, head_dim)`，只读
* `kv_sf`：输入指针（FP32），shape `(seq_len_kv,)`，只读
* `weights`：输入指针（FP32），shape `(M, num_heads)`，只读
* `cu_seqlen_ks`：输入指针（INT32），shape `(M,)`，只读
* `cu_seqlen_ke`：输入指针（INT32），shape `(M,)`，只读
* `logits`：输出指针（FP32），shape `(M, seq_len_kv)`，必须写入结果
* `M`：query token 数量（int64）
* `seq_len_kv`：kv 序列长度（int64）
* `num_heads`：head 数（int64）
* `head_dim`：head 维度（int64）

`run_kernel` 内部需要完成以下工作：

* 根据张量尺寸计算合适的 launch 配置
* launch 你实现的 CUDA kernel，完成 logits 计算
