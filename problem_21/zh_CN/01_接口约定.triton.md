你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(
    q,  # Tensor[float8_e4m3fn]
    kv,  # Tensor[float8_e4m3fn]
    kv_sf,  # Tensor[float32]
    weights,  # Tensor[float32]
    cu_seqlen_ks,  # Tensor[int32]
    cu_seqlen_ke,  # Tensor[int32]
    logits,  # Tensor[float32]
    M,  # int64
    seq_len_kv,  # int64
    num_heads,  # int64
    head_dim,  # int64
):
    ...
```

### 参数说明

* `q`：输入 tensor，shape `(M, num_heads, head_dim)`，类型为 `float8_e4m3fn`
* `kv`：输入 tensor，shape `(seq_len_kv, head_dim)`，类型为 `float8_e4m3fn`
* `kv_sf`：输入 tensor，shape `(seq_len_kv,)`，类型为 `float32`
* `weights`：输入 tensor，shape `(M, num_heads)`，类型为 `float32`
* `cu_seqlen_ks`：输入 tensor，shape `(M,)`，类型为 `int32`
* `cu_seqlen_ke`：输入 tensor，shape `(M,)`，类型为 `int32`
* `logits`：输出 tensor，shape `(M, seq_len_kv)`，类型为 `float32`
* `M`、`seq_len_kv`、`num_heads`、`head_dim`：对应的尺寸参数

`run_kernel` 内部需要完成以下工作：

* 根据张量尺寸计算合适的 launch 配置
* 调用你实现的 Triton kernel，或使用等价的 GPU Tensor 算子，完成 logits 计算
