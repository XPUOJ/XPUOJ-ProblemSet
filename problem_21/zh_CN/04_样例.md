## 输入

设：

```text
M = 2
seq_len_kv = 4
num_heads = 2
head_dim = 2
```

则：

* `q` 的形状为 `(2, 2, 2)`
* `kv` 的形状为 `(4, 2)`
* `kv_sf` 的形状为 `(4,)`
* `weights` 的形状为 `(2, 2)`
* `cu_seqlen_ks` 与 `cu_seqlen_ke` 的形状均为 `(2,)`

例如可以有如下 attention window：

```text
cu_seqlen_ks = [0, 1]
cu_seqlen_ke = [4, 3]
```

这表示：

* 第 0 个 query 位置可访问 kv 的 `[0, 4)` 区间
* 第 1 个 query 位置可访问 kv 的 `[1, 3)` 区间

## 输出

输出 `logits` 的形状为 `(2, 4)`。

对于第 0 个 query 位置，会对全部 4 个 kv 位置计算：

```text
score[h, 0, n] = relu(sum_d(float(q[0, h, d]) * (float(kv[n, d]) * kv_sf[n])))
logits[0, n] = sum_h(score[h, 0, n] * weights[0, h])
```

对于第 1 个 query 位置，只有 `n in [1, 3)` 保留，其余位置写成 `-inf`：

```text
logits[1, 0] = -inf
logits[1, 1] = valid score
logits[1, 2] = valid score
logits[1, 3] = -inf
```
