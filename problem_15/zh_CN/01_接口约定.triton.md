你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序必须完全一致：

```python
import torch
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(x, w_all, out, out_dims, batch_size, in_features, K):
    ...
```

### 参数说明

* `x`：输入 tensor（bfloat16），shape $(batch\_size, in\_features)$，行优先连续存储，只读
* `w_all`：输入 tensor（bfloat16），shape $(total\_out, in\_features)$，拼接后的权重矩阵
* `out`：输出 tensor（bfloat16），shape $(batch\_size, total\_out)$，需写入结果
* `out_dims`：输入 tensor（int64），shape $(K,)$，每路输出的维度
* `batch_size`：批大小
* `in_features`：输入特征数
* `K`：输出路数

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* 对每路 $i$，计算对应的矩阵乘法
