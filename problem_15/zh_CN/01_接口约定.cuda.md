你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_bf16.h>

extern "C" void run_kernel(
    const __nv_bfloat16* x,
    const __nv_bfloat16* w_all,
    __nv_bfloat16* out,
    const int64_t* out_dims,
    int64_t batch_size,
    int64_t in_features,
    int64_t K
);
```

### 参数说明

* `x`：输入指针（bfloat16），shape $(batch\_size, in\_features)$，行优先连续存储，只读
* `w_all`：输入指针（bfloat16），shape $(total\_out, in\_features)$，其中 $total\_out = out\_dims[0] + ... + out\_dims[K-1]$，只读
* `out`：输出指针（bfloat16），shape $(batch\_size, total\_out)$，行优先连续存储，需写入计算结果
* `out_dims`：输入指针（int64），shape $(K,)$，$out\_dims[i]$ 为第 $i$ 路输出的维度，只读
* `batch_size`：批大小（int64）
* `in_features`：输入特征数（int64）
* `K`：输出路数（int64）

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* 对每路 $i$（$0 \le i < K$），令 $offset_i = out\_dims[0]+...+out\_dims[i-1]$，计算 $out[:, offset_i : offset_i+out\_dims[i]] = x \times w\_all[offset_i : offset_i+out\_dims[i], :]^T$
