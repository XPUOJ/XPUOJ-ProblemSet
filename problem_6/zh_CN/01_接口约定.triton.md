你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(
    x,  # Tensor[bfloat16]
    y,  # Tensor[bfloat16]
    T,  # int64
    H,  # int64
):
    ...
```

### 参数说明

* `x`：输入 tensor（bfloat16），shape $(T, 2H)$，行优先连续存储，只读
* `y`：输出 tensor（bfloat16），shape $(T, H)$，行优先连续存储，需写入结果
* `T`：序列长度
* `H`：隐藏层维度

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 triton kernel，完成 GeGLU 计算
