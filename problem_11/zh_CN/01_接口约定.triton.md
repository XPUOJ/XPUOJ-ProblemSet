你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序必须完全一致：

```python
import torch
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(A, B, m_indices, D, M_total, K, N, num_groups):
    ...
```

### 参数说明

* `A`：输入 tensor（bfloat16），shape $(M_{total}, K)$，行优先连续存储，只读
* `B`：输入 tensor（bfloat16），shape $(num\_groups, N, K)$，每个组的权重矩阵
* `m_indices`：输入 tensor（int32），shape $(M_{total},)$，每行的组索引
* `D`：输出 tensor（bfloat16），shape $(M_{total}, N)$，需写入结果
* `M_total`：A 和 D 的第一维大小
* `K`：A 和 B 的最后一维大小
* `N`：B 的第二维大小，D 的第二维大小
* `num_groups`：分组数量

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 triton kernel，完成分组矩阵乘法
