## 输入

`probs`（shape 为 $(2, 4)$）：

```
[[0.1, 0.2, 0.4, 0.3],
 [0.25, 0.25, 0.25, 0.25]]
```

`top_k`（shape 为 $(2,)$）：

```
[2, 3]
```

## 输出

```
[[0.0, 0.0, 0.5714, 0.4286],  
 [0.3333, 0.3333, 0.3333, 0.0]]
```

## 计算过程说明

**第 1 行**（`top_k = 2`）：

该行的概率值为 `[0.1, 0.2, 0.4, 0.3]`，其中最大的两个值是 `0.4`（索引 2）和 `0.3`（索引 3）。将其他位置置零后得到 `[0.0, 0.0, 0.4, 0.3]`，保留的概率之和为 `0.7`，重归一化后得到 `[0.0, 0.0, 0.4/0.7 ≈ 0.5714, 0.3/0.7 ≈ 0.4286]`。

**第 2 行**（`top_k = 3`）：

该行的概率值均为 `0.25`，最大的三个值是前三个 `0.25`（索引 0、1、2）。将索引 3 置零后得到 `[0.25, 0.25, 0.25, 0.0]`，保留的概率之和为 `0.75`，重归一化后每个保留的概率变为 `0.25/0.75 ≈ 0.3333`，结果为 `[0.3333, 0.3333, 0.3333, 0.0]`。
