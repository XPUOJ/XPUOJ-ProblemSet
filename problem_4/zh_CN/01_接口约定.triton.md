你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序必须完全一致：

```python
import torch
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(probs, top_k, renorm_probs, batch_size, num_classes):
    ...
```

### 参数说明

* `probs`：输入 tensor（float32），shape $(batch\_size, num\_classes)$，行优先连续存储，只读
* `top_k`：输入 tensor（int32），shape $(batch\_size,)$，连续存储，每个样本的 top-k 值
* `renorm_probs`：输出 tensor（float32），shape $(batch\_size, num\_classes)$，行优先连续存储，需写入结果
* `batch_size`：批次大小
* `num_classes`：类别数量

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 triton kernel，完成 Top-k 重归一化计算
