你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>

extern "C" void run_kernel(
    const float* probs,
    const int32_t* top_k,
    float* renorm_probs,
    int64_t batch_size,
    int64_t num_classes
);
```

### 参数说明

* `probs`：输入指针（float32），shape $(batch\_size, num\_classes)$，行优先连续存储，只读
* `top_k`：输入指针（int32），shape $(batch\_size,)$，连续存储，每个样本的 top-k 值，取值范围 $(0, num\_classes)$
* `renorm_probs`：输出指针（float32），shape $(batch\_size, num\_classes)$，行优先连续存储，需写入重归一化后的结果
* `batch_size`：批次大小（int64）
* `num_classes`：类别数量（int64）

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 CUDA `__global__` kernel，完成 Top-k 重归一化计算
