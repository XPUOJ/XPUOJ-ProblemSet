## 输入

`x`（shape 为 $(2, 3)$，其中 $T=2, H=3$）：

```
[[1.0, -2.0, 3.0],
 [-1.0, 0.0, 2.0]]
```

## 输出

`y`（shape 为 $(2, 3)$）：

```
[[1.0, 0.0, 3.0],
 [0.0, 0.0, 2.0]]
```

## 计算过程说明

ReLU 函数对输入矩阵的每个元素独立进行计算：

* $x_{0,0} = 1.0$（正值）→ $y_{0,0} = 1.0$
* $x_{0,1} = -2.0$（负值）→ $y_{0,1} = 0.0$
* $x_{0,2} = 3.0$（正值）→ $y_{0,2} = 3.0$
* $x_{1,0} = -1.0$（负值）→ $y_{1,0} = 0.0$
* $x_{1,1} = 0.0$（零值）→ $y_{1,1} = 0.0$
* $x_{1,2} = 2.0$（正值）→ $y_{1,2} = 2.0$
