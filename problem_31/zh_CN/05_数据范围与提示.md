## 数据范围

* `A.dtype == torch.float8_e4m3fn`
* `B.dtype == torch.float8_e4m3fn`
* `A_scale.dtype == torch.float32`
* `B_scale.dtype == torch.float32`
* `output.dtype == torch.bfloat16`

## 正确性要求

* 必须正确按 `BLOCK_SIZE` 对 `K` 维做分块
* 必须正确使用 `A_scale[i, k_block]`
* 必须正确使用 `B_scale[n_block, k_block]`
* 必须先完成反量化，再进行矩阵乘法
* 必须将结果以 `bfloat16` 写入 `output`

## 提示

* `A` 的缩放方式是 **每行、按 K-block 缩放**
* `B` 的缩放方式是 **按 (N-block, K-block) 二维缩放**
* 你可以先在 `float32` 中完成反量化与累加，再将结果转换成 `bfloat16`
* 当 `N` 或 `K` 不是 `BLOCK_SIZE` 的整数倍时，尾块需要正确处理边界

## 评测流程

评测程序会先进行预热，再重复执行多轮取平均时间。
注意：为了保证计时准确，**不建议你在 `run_kernel` 内部做 `cudaDeviceSynchronize()` 或显式同步**。
