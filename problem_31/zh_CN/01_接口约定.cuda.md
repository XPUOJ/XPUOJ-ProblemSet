你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_fp8.h>
#include <cuda_bf16.h>

extern "C" void run_kernel(
    const __nv_fp8_e4m3* A,
    const float* A_scale,
    const __nv_fp8_e4m3* B,
    const float* B_scale,
    __nv_bfloat16* output,
    int64_t M,
    int64_t N,
    int64_t K,
    int64_t BLOCK_SIZE
);
```

### 参数说明

* `A`：输入指针，shape `(M, K)`，类型为 FP8 E4M3
* `A_scale`：输入指针，shape `(M, K_blocks)`，类型为 float32
* `B`：输入指针，shape `(N, K)`，类型为 FP8 E4M3
* `B_scale`：输入指针，shape `(N_blocks, K_blocks)`，类型为 float32
* `output`：输出指针，shape `(M, N)`，类型为 bfloat16
* `M`：输出矩阵行数
* `N`：输出矩阵列数
* `K`：归约维长度
* `BLOCK_SIZE`：groupwise scaling 的分块大小

其中：

```text
K_blocks = ceil_div(K, BLOCK_SIZE)
N_blocks = ceil_div(N, BLOCK_SIZE)
```

`run_kernel` 内部需要完成以下工作：

* 根据 `M / N / K / BLOCK_SIZE` 计算 scale 的索引方式
* 自行计算合适的 launch 配置
* launch 你实现的 CUDA kernel，完成反量化与 GEMM 计算
