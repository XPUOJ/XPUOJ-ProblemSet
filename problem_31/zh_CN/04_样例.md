## 输入

```text
M = 2
N = 2
K = 4
BLOCK_SIZE = 2
```

此时：

```text
K_blocks = ceil_div(4, 2) = 2
N_blocks = ceil_div(2, 2) = 1
```

所以：

* `A_scale` 的形状为 `(2, 2)`
* `B_scale` 的形状为 `(1, 2)`

此时 `A` 的每一行都会沿 `K` 维分成两个 block：

* 第 0 个 block 覆盖 `k in [0, 2)`
* 第 1 个 block 覆盖 `k in [2, 4)`

因此：

```text
A_scaled[i, 0] = float(A[i, 0]) * A_scale[i, 0]
A_scaled[i, 1] = float(A[i, 1]) * A_scale[i, 0]
A_scaled[i, 2] = float(A[i, 2]) * A_scale[i, 1]
A_scaled[i, 3] = float(A[i, 3]) * A_scale[i, 1]
```

对于 `B`，由于 `N_blocks = 1`，所有输出列都来自同一个 `N` 方向 block，因此 `B_scale` 只在 `K` 方向区分两个 block：

```text
B_scaled[j, 0] = float(B[j, 0]) * B_scale[0, 0]
B_scaled[j, 1] = float(B[j, 1]) * B_scale[0, 0]
B_scaled[j, 2] = float(B[j, 2]) * B_scale[0, 1]
B_scaled[j, 3] = float(B[j, 3]) * B_scale[0, 1]
```

最后输出矩阵按如下方式计算：

```text
output[0, 0] = bfloat16(sum_k(A_scaled[0, k] * B_scaled[0, k]))
output[0, 1] = bfloat16(sum_k(A_scaled[0, k] * B_scaled[1, k]))
output[1, 0] = bfloat16(sum_k(A_scaled[1, k] * B_scaled[0, k]))
output[1, 1] = bfloat16(sum_k(A_scaled[1, k] * B_scaled[1, k]))
```
