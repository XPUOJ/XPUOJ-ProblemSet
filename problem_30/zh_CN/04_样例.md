## 输入

```text
N = 1
C_in = 1
C_out = 1
D = H = W = 3
T = R = S = 3
stride = 1
padding = 0
```

此时：

```text
D_out = (3 + 0 - 3) / 1 + 1 = 1
H_out = (3 + 0 - 3) / 1 + 1 = 1
W_out = (3 + 0 - 3) / 1 + 1 = 1
```

所以输出张量形状为：

```text
output.shape = (1, 1, 1, 1, 1)
```

该唯一位置的输出值，等于输入体与卷积核对应位置逐元素乘加后，再加上偏置：

```text
output[0, 0, 0, 0, 0] =
    sum_{k_d, k_h, k_w} x[0, 0, k_d, k_h, k_w] * weight[0, 0, k_d, k_h, k_w]
    + bias[0]
```
