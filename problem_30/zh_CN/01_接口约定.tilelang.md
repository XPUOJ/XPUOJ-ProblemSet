你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def conv3d_fwd(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    x,  # Tensor[float32]
    weight,  # Tensor[float32]
    bias,  # Tensor[float32]
    output,  # Tensor[float32]
    N,  # int64
    C_in,  # int64
    C_out,  # int64
    D,  # int64
    H,  # int64
    W,  # int64
    T,  # int64
    R,  # int64
    S,  # int64
    stride,  # int64
    padding,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = conv3d_fwd(...)
    real_kernel(x, weight, bias, output, N, C_in, C_out, D, H, W, T, R, S, stride, padding)
```

### 参数说明

* `x`：输入 tensor，shape `(N, C_in, D, H, W)`，类型为 `float32`
* `weight`：输入 tensor，shape `(C_out, C_in, T, R, S)`，类型为 `float32`
* `bias`：输入 tensor，shape `(C_out,)`，类型为 `float32`
* `output`：输出 tensor，shape `(N, C_out, D_out, H_out, W_out)`，类型为 `float32`
* 其余参数含义与 CUDA 接口一致

`run_kernel` 内部需要完成以下工作：

* 根据张量尺寸计算输出形状
* 调用你实现的 TileLang kernel，或使用等价的 GPU Tensor 算子，完成 3D 卷积前向传播
