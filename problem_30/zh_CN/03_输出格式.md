输出张量满足：

```text
D_out = floor((D + 2 * padding - T) / stride + 1)
H_out = floor((H + 2 * padding - R) / stride + 1)
W_out = floor((W + 2 * padding - S) / stride + 1)

output[n, c_out, d, h, w] =
    sum_{c_in, k_d, k_h, k_w}
        x[n, c_in, d * stride - padding + k_d, h * stride - padding + k_h, w * stride - padding + k_w]
        * weight[c_out, c_in, k_d, k_h, k_w]
    + bias[c_out]
```
