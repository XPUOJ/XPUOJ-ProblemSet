你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(
    values,  # Tensor[dtype 见参数说明]
    indices,  # Tensor[int64]
    output,  # Tensor[dtype 见参数说明]
    first_axis_dim,  # int64
    M,  # int64
    H,  # int64
    dtype_code,  # int64
):
    ...
```

### 参数说明

* `values`：输入 tensor，shape `(M, H)`，数据类型由 `dtype_code` 指定
* `indices`：输入 tensor，shape `(M,)`，类型为 `int64`
* `output`：输出 tensor，shape `(first_axis_dim, H)`，类型与 `values` 相同
* `first_axis_dim`：输出第 0 维大小
* `M`：待写入的行数
* `H`：每行元素数
* `dtype_code`：数据类型编码，`0/1/2` 分别对应 `bfloat16/float16/float32`

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 launch 配置
* 调用你实现的 Triton kernel，或使用等价的 GPU Tensor 算子，完成沿第 0 维的 scatter 写入
