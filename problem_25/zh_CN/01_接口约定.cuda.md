你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>

extern "C" void run_kernel(
    const void* values,
    const int64_t* indices,
    void* output,
    int64_t first_axis_dim,
    int64_t M,
    int64_t H,
    int64_t dtype_code
);
```

### 参数说明

* `values`：输入指针，shape `(M, H)`
* `indices`：输入指针，shape `(M,)`，类型为 INT64
* `output`：输出指针，shape `(first_axis_dim, H)`
* `first_axis_dim`：输出第 0 维大小
* `M`：待写入的行数
* `H`：每行元素数
* `dtype_code`：数据类型编码
  - `0` 表示 `bfloat16`
  - `1` 表示 `float16`
  - `2` 表示 `float32`

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 launch 配置
* launch 你实现的 CUDA kernel，完成沿第 0 维的 scatter 写入
