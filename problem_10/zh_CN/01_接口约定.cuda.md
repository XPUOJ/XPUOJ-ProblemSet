你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_bf16.h>
#include <cuda_fp8.h>

extern "C" void run_kernel(
    const __nv_fp8_e4m3* A,
    const float* A_scale,
    const __nv_fp8_e4m3* B,
    const float* B_scale,
    __nv_bfloat16* D,
    int64_t M,
    int64_t K,
    int64_t N
);
```

### 参数说明

* `A`：输入指针（FP8E4M3），shape $(M, K)$，行优先连续存储，只读
* `A_scale`：输入指针（float32），shape $(M, \lceil K/128 \rceil)$，行优先连续存储，A 矩阵的 1D 缩放因子
* `B`：输入指针（FP8E4M3），shape $(N, K)$，行优先连续存储，只读
* `B_scale`：输入指针（float32），shape $(\lceil N/128 \rceil, \lceil K/128 \rceil)$，行优先连续存储，B 矩阵的 2D 缩放因子
* `D`：输出指针（bfloat16），shape $(M, N)$，行优先连续存储，需写入计算结果
* `M`：A 的第一维大小（int64）
* `K`：A 的第二维和 B 的第二维大小（int64）
* `N`：B 的第一维大小（int64）

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 CUDA `__global__` kernel，完成 FP8 反量化和矩阵乘法
