你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def fp8_gemm(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    A,  # Tensor[float8_e4m3fn]
    A_scale,  # Tensor[float32]
    B,  # Tensor[float8_e4m3fn]
    B_scale,  # Tensor[float32]
    D,  # Tensor[bfloat16]
    M,  # int64
    K,  # int64
    N,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = fp8_gemm(...)
    real_kernel(A, A_scale, B, B_scale, D, M, K, N)
```

### 参数说明

* `A`：输入 tensor（FP8E4M3），shape $(M, K)$，行优先连续存储，只读
* `A_scale`：输入 tensor（float32），shape $(M, \lceil K/128 \rceil)$，A 矩阵的 1D 缩放因子
* `B`：输入 tensor（FP8E4M3），shape $(N, K)$，行优先连续存储，只读
* `B_scale`：输入 tensor（float32），shape $(\lceil N/128 \rceil, \lceil K/128 \rceil)$，B 矩阵的 2D 缩放因子
* `D`：输出 tensor（bfloat16），shape $(M, N)$，需写入计算结果
* `M`：A 的第一维大小
* `K`：A 的第二维和 B 的第二维大小
* `N`：B 的第一维大小

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 tilelang kernel，完成 FP8 反量化和矩阵乘法
