你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序必须完全一致：

```python
import torch
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(input, out, batch_size, hidden_size):
    ...
```

### 参数说明

* `input`：输入 tensor（bfloat16），shape $(\text{batch_size}, 2 \cdot \text{hidden_size})$，行优先连续存储，只读
* `out`：输出 tensor（bfloat16），shape $(\text{batch_size}, \text{hidden_size})$，需写入结果
* `batch_size`：第一维大小
* `hidden_size`：隐藏层大小

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 triton kernel，完成融合 SiLU + Mul 计算
