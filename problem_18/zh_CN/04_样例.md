## 输入

`input`（shape $(2, 4)$，其中 $\text{batch_size}=2$，$\text{hidden_size}=2$）：

```
[[1.0, 2.0, 3.0, 4.0],
 [0.5, 1.5, 2.5, 3.5]]
```

该输入将在最后一维被分为两部分：
* $x_1$（前半部分）：`[[1.0, 2.0], [0.5, 1.5]]`
* $x_2$（后半部分）：`[[3.0, 4.0], [2.5, 3.5]]`

## 输出

`out`（shape $(2, 2)$）：

**计算过程**：

对于位置 $(0, 0)$：
* $\text{SiLU}(1.0) = 1.0 / (1 + e^{-1.0}) \approx 0.731$
* $y_{0,0} = 0.731 \times 3.0 \approx 2.193$

对于位置 $(0, 1)$：
* $\text{SiLU}(2.0) = 2.0 / (1 + e^{-2.0}) \approx 1.762$
* $y_{0,1} = 1.762 \times 4.0 \approx 7.048$

对于位置 $(1, 0)$：
* $\text{SiLU}(0.5) = 0.5 / (1 + e^{-0.5}) \approx 0.311$
* $y_{1,0} = 0.311 \times 2.5 \approx 0.778$

对于位置 $(1, 1)$：
* $\text{SiLU}(1.5) = 1.5 / (1 + e^{-1.5}) \approx 1.154$
* $y_{1,1} = 1.154 \times 3.5 \approx 4.039$

**输出结果**：

```
[[2.193, 7.048],
 [0.778, 4.039]]
```
（实际输出为 bfloat16，数值可能有精度差异）
