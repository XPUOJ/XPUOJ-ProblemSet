你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(
    a,  # Tensor[int8]
    b,  # Tensor[int8]
    output,  # Tensor[int32]
    M,  # int64
    N,  # int64
    K,  # int64
):
    ...
```

### 参数说明

* `a`：输入 tensor，shape `(M, K)`，类型为 `int8`
* `b`：输入 tensor，shape `(K, N)`，类型为 `int8`
* `output`：输出 tensor，shape `(M, N)`，类型为 `int32`
* `M`、`N`、`K`：矩阵乘法尺寸

`run_kernel` 内部需要完成以下工作：

* 根据张量尺寸计算合适的 launch 配置
* 调用你实现的 Triton kernel，或使用等价的 GPU Tensor 算子，完成 GEMM
