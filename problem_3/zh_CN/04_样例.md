## 输入

`probs`（shape 为 $(2, 4)$）：

```
[[0.4, 0.2, 0.3, 0.1],
 [0.3, 0.3, 0.1, 0.3]]
```

`top_p`（shape 为 $(2,)$）：

```
[0.8, 0.9]
```

## 输出

```
[[0.5714, 0.0, 0.4286, 0.0], 
 [0.333, 0.333, 0.0, 0.333]]
```

## 计算过程说明

**第 1 行**（`top_p = 0.8`）：

首先将概率降序排序得到 `[0.4, 0.3, 0.2, 0.1]`，计算前缀累积和为 `[0.4, 0.7, 0.9, 1.0]`。由于累积和 `0.7 <= 0.8` 但 `0.9 > 0.8`，因此保留概率值 `0.4` 和 `0.3`，丢弃 `0.2` 和 `0.1`。保留的概率之和为 `0.7`，重归一化后得到 `0.4/0.7 ≈ 0.5714` 和 `0.3/0.7 ≈ 0.4286`，写回原始索引位置后结果为 `[0.5714, 0.0, 0.4286, 0.0]`。

**第 2 行**（`top_p = 0.9`）：

将概率降序排序得到 `[0.3, 0.3, 0.3, 0.1]`，前缀累积和为 `[0.3, 0.6, 0.9, 1.0]`。前三个 `0.3` 的累积和正好等于 `0.9`，因此保留这三项，丢弃 `0.1`。保留的概率之和为 `0.9`，重归一化后每个保留的概率变为 `0.3/0.9 ≈ 0.3333`，写回原始索引位置后结果为 `[0.3333, 0.3333, 0.0, 0.3333]`。
