你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序必须完全一致：

```python
import torch
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(x, w1, w3, group_sizes, out, total_tokens, in_features, out_features, num_groups):
    ...
```

### 参数说明

* `x`：输入 tensor（bfloat16），shape $(total\_tokens, in\_features)$，行优先连续存储，只读
* `w1`：输入 tensor（bfloat16），shape $(num\_groups, out\_features, in\_features)$
* `w3`：输入 tensor（bfloat16），shape $(num\_groups, out\_features, in\_features)$
* `group_sizes`：输入 tensor（int32），shape $(num\_groups,)$，每组 token 数
* `out`：输出 tensor（bfloat16），shape $(total\_tokens, out\_features \times 2)$，需写入结果
* `total_tokens`：总 token 数
* `in_features`：输入特征数
* `out_features`：单权输出的特征数
* `num_groups`：组数

`run_kernel` 内部需要按组计算，对每组使用对应的权重与输入进行矩阵乘法。
