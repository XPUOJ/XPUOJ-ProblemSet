你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_bf16.h>

extern "C" void run_kernel(
    const __nv_bfloat16* x,
    const __nv_bfloat16* w1,
    const __nv_bfloat16* w3,
    const int32_t* group_sizes,
    __nv_bfloat16* out,
    int64_t total_tokens,
    int64_t in_features,
    int64_t out_features,
    int64_t num_groups
);
```

### 参数说明

* `x`：输入指针（bfloat16），shape $(total\_tokens, in\_features)$，行优先连续存储，只读
* `w1`：输入指针（bfloat16），shape $(num\_groups, out\_features, in\_features)$，只读
* `w3`：输入指针（bfloat16），shape $(num\_groups, out\_features, in\_features)$，只读
* `group_sizes`：输入指针（int32），shape $(num\_groups,)$，每组 token 数，只读
* `out`：输出指针（bfloat16），shape $(total\_tokens, out\_features \times 2)$，行优先连续存储，需写入计算结果
* `total_tokens`：总 token 数（int64）
* `in_features`：输入特征数（int64）
* `out_features`：单权输出的特征数（int64），输出每 token 为 $out\_features \times 2$
* `num_groups`：组数（int64）

`run_kernel` 内部需要按组计算：对组 $g$，用 $x$ 中对应区间的行与 $concat(w1[g], w3[g])$ 做矩阵乘，写回 $out$ 的对应行区间。
