你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <stdbool.h>

extern "C" void run_kernel(
    const uint8_t* bits,
    uint8_t* packed,
    int64_t num_bits
);
```

### 参数说明

* `bits`：输入指针（底层按 1 字节布尔值存储），shape $(\text{num_bits},)$，只读
* `packed`：输出指针（uint8），shape $(\lceil \text{num_bits} / 8 \rceil,)$，需写入结果
* `num_bits`：输入 bit 数量（int64）

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 CUDA `__global__` kernel，完成 packbits
