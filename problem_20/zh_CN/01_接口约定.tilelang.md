你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def packbits(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    bits,  # Tensor[uint8]
    packed,  # Tensor[uint8]
    num_bits,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = packbits(...)
    real_kernel(bits, packed, num_bits)
```

### 参数说明

* `bits`：输入 tensor（bool），shape $(\text{num_bits},)$，底层按 1 字节存储
* `packed`：输出 tensor（uint8），shape $(\lceil \text{num_bits} / 8 \rceil,)$
* `num_bits`：输入 bit 数量

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 tilelang kernel，完成 packbits
