## 数据范围

* `a.dtype == torch.int8`
* `b.dtype == torch.float8_e4m3fn`
* `output.dtype == torch.float32`
* `a.shape == (G, M, K)`
* `b.shape == (G, K, N)`
* `output.shape == (G, M, N)`

## 正确性要求

* 必须对每个 group 独立完成矩阵乘法
* 必须将 `a` 和 `b` 按浮点数语义参与乘加
* 必须将结果写入 `output`

## 评测流程

评测程序会先进行预热，再重复执行多轮取平均时间。
注意：为了保证计时准确，**不建议你在 `run_kernel` 内部做 `cudaDeviceSynchronize()` 或显式同步**。
