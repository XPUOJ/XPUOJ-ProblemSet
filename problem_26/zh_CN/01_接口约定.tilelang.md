你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def int4_fp8_grouped_gemm(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    a,  # Tensor[int8]
    b,  # Tensor[float8_e4m3fn]
    output,  # Tensor[float32]
    G,  # int64
    M,  # int64
    N,  # int64
    K,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = int4_fp8_grouped_gemm(...)
    real_kernel(a, b, output, G, M, N, K)
```

### 参数说明

* `a`：输入 tensor，shape `(G, M, K)`，类型为 `int8`
* `b`：输入 tensor，shape `(G, K, N)`，类型为 `float8_e4m3fn`
* `output`：输出 tensor，shape `(G, M, N)`，类型为 `float32`
* `G`、`M`、`N`、`K`：对应的尺寸参数

`run_kernel` 内部需要完成以下工作：

* 根据张量尺寸计算合适的 launch 配置
* 调用你实现的 TileLang kernel，或使用等价的 GPU Tensor 算子，完成 grouped GEMM
