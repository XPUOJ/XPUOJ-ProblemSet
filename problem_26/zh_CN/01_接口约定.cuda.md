你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_fp8.h>

extern "C" void run_kernel(
    const int8_t* a,
    const __nv_fp8_e4m3* b,
    float* output,
    int64_t G,
    int64_t M,
    int64_t N,
    int64_t K
);
```

### 参数说明

* `a`：输入指针，shape `(G, M, K)`，类型为 int8，数值范围位于 `[-8, 7]`
* `b`：输入指针，shape `(G, K, N)`，类型为 FP8 E4M3
* `output`：输出指针，shape `(G, M, N)`，类型为 float32
* `G`：group 数量
* `M`、`N`、`K`：矩阵乘法尺寸

`run_kernel` 内部需要完成以下工作：

* 根据张量尺寸计算合适的 launch 配置
* launch 你实现的 CUDA kernel，完成每个 group 的 GEMM 计算
