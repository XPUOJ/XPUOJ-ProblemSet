## 输入

假设以下参数：
* $\text{batch_size} = 1$，$\text{dim} = 1$，$\text{seq_len} = 4$，$\text{kernel_size} = 3$
* $\text{has_initial_state} = 0$，$\text{silu_activation} = 0$

`x`（shape $(1, 1, 4)$）：

```
[[[1.0, 2.0, 3.0, 4.0]]]
```

`weight`（shape $(1, 3)$）：

```
[[0.1, 0.2, 0.3]]
```

`bias`（shape $(1,)$）：

```
[0.0]
```

## 输出

`output`（shape $(1, 1, 4)$）：

**计算过程**（Causal 卷积，只依赖当前及之前输入）：

对于 $t = 0$（第一个时间步）：
* $\text{src_t}$ 取值：$0 - 2 + 0 = -2$，$0 - 2 + 1 = -1$，$0 - 2 + 2 = 0$
* 只有 $\text{src_t} = 0 \ge 0$ 有效
* $\text{acc} = 0.0 + x[0,0,0] \cdot 0.3 = 1.0 \times 0.3 = 0.3$
* $\text{output}[0,0,0] = 0.3$

对于 $t = 1$：
* $\text{src_t}$ 取值：$-1$，$0$，$1$
* 有效输入：$x[0,0,0]$ 和 $x[0,0,1]$
* $\text{acc} = 0.0 + x[0,0,0] \cdot 0.2 + x[0,0,1] \cdot 0.3 = 1.0 \times 0.2 + 2.0 \times 0.3 = 0.2 + 0.6 = 0.8$
* $\text{output}[0,0,1] = 0.8$

对于 $t = 2$：
* $\text{src_t}$ 取值：$0$，$1$，$2$
* 全部有效
* $\text{acc} = 0.0 + 1.0 \times 0.1 + 2.0 \times 0.2 + 3.0 \times 0.3 = 0.1 + 0.4 + 0.9 = 1.4$
* $\text{output}[0,0,2] = 1.4$

对于 $t = 3$：
* $\text{acc} = 0.0 + 2.0 \times 0.1 + 3.0 \times 0.2 + 4.0 \times 0.3 = 0.2 + 0.6 + 1.2 = 2.0$
* $\text{output}[0,0,3] = 2.0$

**输出结果**：

```
[[[0.3, 0.8, 1.4, 2.0]]]
```
