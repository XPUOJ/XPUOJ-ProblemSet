你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序必须完全一致：

```python
import torch
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(x, weight, bias, conv_states, output, batch_size, dim, seq_len, kernel_size, has_initial_state, silu_activation):
    ...
```

### 参数说明

* `x`：输入 tensor（float32），shape $(\text{batch_size}, \text{dim}, \text{seq_len})$
* `weight`：权重 tensor（float32），shape $(\text{dim}, \text{kernel_size})$
* `bias`：偏置 tensor（float32），shape $(\text{dim},)$
* `conv_states`：初始状态 tensor（float32），shape $(\text{batch_size}, \text{dim}, \text{kernel_size} - 1)$
* `output`：输出 tensor（float32），shape $(\text{batch_size}, \text{dim}, \text{seq_len})$
* `batch_size`：batch 大小
* `dim`：通道数
* `seq_len`：序列长度
* `kernel_size`：卷积核宽度
* `has_initial_state`：是否使用初始状态（0 或 1）
* `silu_activation`：是否应用 SiLU 激活（0 或 1）

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 triton kernel，完成 causal conv1d forward
