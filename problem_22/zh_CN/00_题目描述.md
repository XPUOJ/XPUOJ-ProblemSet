你需要提交一份代码，实现一个 per-token FP8 量化算子：

> **output_q, scale = per_token_quant_fp8(input)**

该算子对输入矩阵的每一行分别计算一个缩放因子，再将该行量化为 `FP8 E4M3`。

对于第 `i` 行：

1. `amax = max(abs(input[i, :]))`
2. `scale[i] = max(amax / 448.0, 1e-12)`
3. `scale_inv = 1.0 / scale[i]`
4. `output_q[i, j] = to_fp8_e4m3fn(clamp(input[i, j] * scale_inv, -448.0, 448.0))`

评测程序会将你的源码编译为 **动态链接库（.so）**，并在 GPU 上通过 FFI 调用你导出的函数：

* `run_kernel`：由你在函数内部计算合适的 launch 配置并完成 per-token FP8 量化

评测同时会进行：

* **性能计时**
* **正确性校验**

如何提交代码详见[评测指南](/d/2)。
