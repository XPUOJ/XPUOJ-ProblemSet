## 输入

设：

```text
M = 2
N = 4
dtype_code = 0
```

输入矩阵：

```text
input =
[
  [1.0, -2.0, 3.0, -4.0],
  [0.5, -1.0, 1.5, -2.0]
]
```

## 输出

第 0 行：

```text
amax = 4.0
scale[0] = max(4.0 / 448.0, 1e-12)
output_q[0, :] = to_fp8_e4m3fn(clamp(input[0, :] / scale[0], -448.0, 448.0))
```

第 1 行：

```text
amax = 2.0
scale[1] = max(2.0 / 448.0, 1e-12)
output_q[1, :] = to_fp8_e4m3fn(clamp(input[1, :] / scale[1], -448.0, 448.0))
```

因此输出为：

```text
scale = [4.0 / 448.0, 2.0 / 448.0]
output_q 为按行独立量化后的 FP8 E4M3 矩阵
```
