你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_fp8.h>

extern "C" void run_kernel(
    const void* input,
    __nv_fp8_storage_t* output_q,
    float* scale,
    int64_t M,
    int64_t N,
    int64_t dtype_code
);
```

### 参数说明

* `input`：输入指针，shape `(M, N)`，只读
* `output_q`：输出指针（FP8 E4M3 存储），shape `(M, N)`，必须写入量化结果
* `scale`：输出指针（FP32），shape `(M,)`，必须写入每一行的缩放因子
* `M`：输入行数（int64）
* `N`：输入列数（int64）
* `dtype_code`：输入类型编码
  - `0` 表示 `bfloat16`
  - `1` 表示 `float16`

`run_kernel` 内部需要完成以下工作：

* 根据矩阵尺寸计算合适的 launch 配置
* launch 你实现的 CUDA kernel，完成逐行动态量化
