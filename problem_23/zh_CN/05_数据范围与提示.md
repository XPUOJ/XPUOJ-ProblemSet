## 数据范围

* `input.dtype == torch.bfloat16`
* `indices.dtype == torch.int64`
* `output.dtype == torch.bfloat16`
* `input.shape == (N, H)`
* `indices.shape == (M,)`
* `output.shape == (M, H)`

## 正确性要求

* 必须严格按 `indices` 沿第 0 维 gather
* 必须完整复制每一行的所有列
