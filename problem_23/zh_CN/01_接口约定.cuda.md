你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_bf16.h>

extern "C" void run_kernel(
    const __nv_bfloat16* input,
    const int64_t* indices,
    __nv_bfloat16* output,
    int64_t N,
    int64_t M,
    int64_t H
);
```

### 参数说明

* `input`：输入指针（bfloat16），shape `(N, H)`，行优先连续存储，只读
* `indices`：输入指针（int64），shape `(M,)`，只读
* `output`：输出指针（bfloat16），shape `(M, H)`，行优先连续存储，需写入结果
* `N`：输入第 0 维大小
* `M`：索引长度
* `H`：每行元素数

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 launch 配置
* launch 你实现的 CUDA kernel，完成沿第 0 维的 gather
