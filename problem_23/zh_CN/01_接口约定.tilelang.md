你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def gather_first_axis(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    input,  # Tensor[bfloat16]
    indices,  # Tensor[int64]
    output,  # Tensor[bfloat16]
    N,  # int64
    M,  # int64
    H,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = gather_first_axis(...)
    real_kernel(input, indices, output, N, M, H)
```

### 参数说明

* `input`：输入 tensor，shape `(N, H)`，类型为 `bfloat16`
* `indices`：输入 tensor，shape `(M,)`，类型为 `int64`
* `output`：输出 tensor，shape `(M, H)`，类型为 `bfloat16`
* `N`：输入第 0 维大小
* `M`：索引长度
* `H`：每行元素数

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 launch 配置
* 调用你实现的 TileLang kernel，或使用等价的 GPU Tensor 算子，完成沿第 0 维的 gather
