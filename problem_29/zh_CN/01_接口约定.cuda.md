你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>

extern "C" void run_kernel(
    const double* a,
    const double* b,
    double* output,
    double alpha,
    int64_t upper,
    int64_t M,
    int64_t N
);
```

### 参数说明

* `a`：输入指针，shape `(M, M)`，类型为 float64
* `b`：输入指针，shape `(M, N)`，类型为 float64
* `output`：输出指针，shape `(M, N)`，类型为 float64
* `alpha`：缩放因子
* `upper`：是否取上三角，`1` 表示上三角，`0` 表示下三角
* `M`、`N`：矩阵尺寸
