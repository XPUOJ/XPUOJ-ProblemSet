你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def trmm(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    a,  # Tensor[float64]
    b,  # Tensor[float64]
    output,  # Tensor[float64]
    alpha,  # float64
    upper,  # int64
    M,  # int64
    N,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = trmm(...)
    real_kernel(a, b, output, alpha, upper, M, N)
```

### 参数说明

* `a`：输入 tensor，shape `(M, M)`，类型为 float64
* `b`：输入 tensor，shape `(M, N)`，类型为 float64
* `output`：输出 tensor，shape `(M, N)`，类型为 float64
* `alpha`：缩放因子
* `upper`：是否取上三角，`1` 表示上三角，`0` 表示下三角
* `M`、`N`：矩阵尺寸
