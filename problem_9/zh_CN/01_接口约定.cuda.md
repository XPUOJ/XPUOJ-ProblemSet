你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>

extern "C" void run_kernel(
    const float* logits,
    const int64_t* labels,
    float* loss,
    int64_t M,
    int64_t N
);
```

### 参数说明

* `logits`：输入指针（float32），shape $(M, N)$，行优先连续存储，只读
* `labels`：输入指针（int64），shape $(M,)$，连续存储，每个元素表示对应样本的真实类别，取值范围 $[0, N)$
* `loss`：输出指针（float32），shape $(M,)$，连续存储，需写入每个样本的交叉熵损失
* `M`：样本数量（int64）
* `N`：类别数量（int64）

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 CUDA `__global__` kernel，完成交叉熵计算
