你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(
    logits,  # Tensor[float32]
    labels,  # Tensor[int64]
    loss,  # Tensor[float32]
    M,  # int64
    N,  # int64
):
    ...
```

### 参数说明

* `logits`：输入 tensor（float32），shape $(M, N)$，行优先连续存储，只读
* `labels`：输入 tensor（int64），shape $(M,)$，连续存储，每个元素表示对应样本的真实类别
* `loss`：输出 tensor（float32），shape $(M,)$，连续存储，需写入每个样本的损失
* `M`：样本数量
* `N`：类别数量

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 triton kernel，完成交叉熵计算
