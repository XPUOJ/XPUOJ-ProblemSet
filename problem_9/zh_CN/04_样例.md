## 输入

$M = 2$，$N = 3$

`logits`：

```
[[1.0, 2.0, 3.0],
 [0.5, 1.5, 0.0]]
```

`labels`：

```
[2, 0]
```

## 输出

`loss`：

```
[0.4076, 1.4076]
```

## 计算过程说明

**样本 0**（标签为 2）：

首先对 logits $[1.0, 2.0, 3.0]$ 进行 softmax：
* $\exp([1.0, 2.0, 3.0]) \approx [2.718, 7.389, 20.086]$
* 求和 $\approx 30.193$
* softmax 结果 $\approx [0.0900, 0.2447, 0.6652]$

取标签 2 位置的概率 $0.6652$，计算负对数：
* $\text{loss}[0] = -\log(0.6652) \approx 0.4076$

**样本 1**（标签为 0）：

对 logits $[0.5, 1.5, 0.0]$ 进行 softmax：
* $\exp([0.5, 1.5, 0.0]) \approx [1.649, 4.482, 1.000]$
* 求和 $\approx 7.131$
* softmax 结果 $\approx [0.2315, 0.6285, 0.1400]$

取标签 0 位置的概率 $0.2315$，计算负对数：
* $\text{loss}[1] = -\log(0.2315) \approx 1.4076$
