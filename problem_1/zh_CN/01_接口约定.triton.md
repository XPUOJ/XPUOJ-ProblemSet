你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(
    A,  # Tensor[fp16]
    B,  # Tensor[fp16]
    numel,  # int64
):
    ...
```

### 参数说明

* `A`：输入/输出tensor（fp16），必须原地写回
* `B`：输入tensor（fp16），只读
* `numel`：元素总数

`run_kernel` 内部需要：

* 自行计算合适的 grid/block
* launch 你实现的 triton kernel 完成 `A += B`
