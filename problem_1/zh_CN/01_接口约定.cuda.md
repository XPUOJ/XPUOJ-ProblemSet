你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_fp16.h>

extern "C" void run_kernel(__half* A, const __half* B, int64_t numel);
```

### 参数说明

* `A`：输入/输出指针（fp16），必须原地写回
* `B`：输入指针（fp16），只读
* `numel`：元素总数（int64）

`run_kernel` 内部需要：

* 自行计算合适的 grid/block
* launch 你实现的 CUDA `__global__` kernel 完成 `A += B`
