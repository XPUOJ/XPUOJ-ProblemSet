你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def add(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    A,  # Tensor[fp16]
    B,  # Tensor[fp16]
    numel,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = add(...)
    real_kernel(A, B, numel)
```

### 参数说明

* `A`：输入/输出tensor（fp16），必须原地写回
* `B`：输入tensor（fp16），只读
* `numel`：元素总数

`run_kernel` 内部需要：

* 自行计算合适的 grid/block
* launch 你实现的 tilelang kernel 完成 `A += B`
