## 输入

```text
N = 1
C_in = 1
C_out = 1
H = W = 3
R = S = 3
stride = 1
padding = 0
```

此时：

```text
H_out = (3 + 0 - 3) / 1 + 1 = 1
W_out = (3 + 0 - 3) / 1 + 1 = 1
```

所以输出张量形状为：

```text
output.shape = (1, 1, 1, 1)
```

该唯一位置的输出值，等于输入窗口与卷积核逐元素乘加后再加上偏置，并最终转换为 int32。
