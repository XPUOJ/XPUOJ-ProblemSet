你需要将计算结果写入 `output`，满足：

```text
H_out = floor((H + 2 * padding - R) / stride + 1)
W_out = floor((W + 2 * padding - S) / stride + 1)

output[n, c_out, h, w] =
    int32(
        sum_{c_in, k_h, k_w}
            x[n, c_in, h * stride - padding + k_h, w * stride - padding + k_w]
            * weight[c_out, c_in, k_h, k_w]
        + bias[c_out]
    )
```
