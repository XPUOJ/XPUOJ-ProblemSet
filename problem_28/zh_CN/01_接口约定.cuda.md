你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_bf16.h>

extern "C" void run_kernel(
    const __nv_bfloat16* a,
    const __nv_bfloat16* b,
    __nv_bfloat16* output,
    int64_t pattern_id,
    int64_t B,
    int64_t M,
    int64_t N,
    int64_t K
);
```

### 参数说明

* `a`：输入指针，shape 由 `pattern_id` 决定，类型为 bfloat16
* `b`：输入指针，shape 由 `pattern_id` 决定，类型为 bfloat16
* `output`：输出指针，shape 固定为 `(B, M, N)`，类型为 bfloat16
* `pattern_id`：模式编号，仅允许取 `1`、`2`、`3`
* `B`、`M`、`N`、`K`：张量收缩尺寸

不同 `pattern_id` 下的张量形状如下：

```text
pattern_id = 1:
  a.shape = (B, M, K)
  b.shape = (B, K, N)
  output.shape = (B, M, N)

pattern_id = 2:
  a.shape = (B, K, M)
  b.shape = (B, K, N)
  output.shape = (B, M, N)

pattern_id = 3:
  a.shape = (B, M, K)
  b.shape = (B, N, K)
  output.shape = (B, M, N)
```

`run_kernel` 内部需要完成以下工作：

* 根据 `pattern_id` 选择正确的索引方式
* 根据张量尺寸计算合适的 launch 配置
* launch 你实现的 CUDA kernel，完成对应的张量收缩
