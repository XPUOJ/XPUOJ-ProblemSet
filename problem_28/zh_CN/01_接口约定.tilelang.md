你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def tensor_contraction(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    a,  # Tensor[bfloat16]
    b,  # Tensor[bfloat16]
    output,  # Tensor[bfloat16]
    pattern_id,  # int64
    B,  # int64
    M,  # int64
    N,  # int64
    K,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = tensor_contraction(...)
    real_kernel(a, b, output, pattern_id, B, M, N, K)
```

### 参数说明

* `a`：输入 tensor，shape 由 `pattern_id` 决定，类型为 `bfloat16`
* `b`：输入 tensor，shape 由 `pattern_id` 决定，类型为 `bfloat16`
* `output`：输出 tensor，shape 固定为 `(B, M, N)`，类型为 `bfloat16`
* `pattern_id`、`B`、`M`、`N`、`K`：含义与 CUDA 接口一致

`run_kernel` 内部需要完成以下工作：

* 根据 `pattern_id` 选择正确的 contraction 公式
* 调用你实现的 TileLang kernel，或使用等价的 GPU Tensor 算子，完成对应的张量收缩
