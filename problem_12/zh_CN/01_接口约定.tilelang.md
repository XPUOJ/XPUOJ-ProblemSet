你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def masked_grouped_gemm(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    a,  # Tensor[bfloat16]
    b,  # Tensor[bfloat16]
    masked_m,  # Tensor[int32]
    d,  # Tensor[bfloat16]
    num_groups,  # int64
    max_m,  # int64
    n,  # int64
    k,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = masked_grouped_gemm(...)
    real_kernel(a, b, masked_m, d, num_groups, max_m, n, k)
```

### 参数说明

* `a`：输入 tensor（bfloat16），shape $(num\_groups, max\_m, k)$，行优先连续存储，只读
* `b`：输入 tensor（bfloat16），shape $(num\_groups, n, k)$，每个组的权重矩阵
* `masked_m`：输入 tensor（int32），shape $(num\_groups,)$，每个组的实际处理行数
* `d`：输出 tensor（bfloat16），shape $(num\_groups, max\_m, n)$，需写入结果
* `num_groups`：分组数量
* `max_m`：每个组的最大行数
* `n`：b 的第二维大小，d 的第三维大小
* `k`：a 和 b 的最后一维大小

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 tilelang kernel，完成带掩码的分组矩阵乘法
