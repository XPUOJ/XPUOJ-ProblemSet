你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(
    input,  # Tensor[bfloat16]
    out,  # Tensor[bfloat16]
    batch_size,  # int64
    hidden_size,  # int64
):
    ...
```

### 参数说明

* `input`：tensor，dtype 为 bfloat16（具体形状与读写语义见数据范围与提示）
* `out`：tensor，dtype 为 bfloat16（具体形状与读写语义见数据范围与提示）
* `batch_size`：标量，类型为 int64
* `hidden_size`：标量，类型为 int64
