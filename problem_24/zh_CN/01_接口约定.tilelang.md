你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def fused_gelu_tanh_mul(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    input,  # Tensor[bfloat16]
    out,  # Tensor[bfloat16]
    batch_size,  # int64
    hidden_size,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = fused_gelu_tanh_mul(...)
    real_kernel(input, out, batch_size, hidden_size)
```

### 参数说明

* `input`：tensor，dtype 为 bfloat16（具体形状与读写语义见数据范围与提示）
* `out`：tensor，dtype 为 bfloat16（具体形状与读写语义见数据范围与提示）
* `batch_size`：标量，类型为 int64
* `hidden_size`：标量，类型为 int64
