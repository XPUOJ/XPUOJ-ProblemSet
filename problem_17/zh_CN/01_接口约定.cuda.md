你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_bf16.h>
#include <cuda_fp8.h>

extern "C" void run_kernel(
    const __nv_bfloat16* input,
    __nv_fp8_storage_t* output_q,
    float* scale,
    int64_t is_static,
    int64_t M,
    int64_t N
);
```

### 参数说明

* `input`：输入指针（bfloat16），shape $(M, N)$，行优先连续存储，只读
* `output_q`：输出指针（FP8 E4M3FN 存储类型），shape $(M, N)$，行优先连续存储，需写入量化结果
* `scale`：输出指针（float32 标量），需写入计算的缩放因子
* `is_static`：int64 标量（0/1），本题按动态 scale 处理（值为 0）
* `M`：输入行数（int64）
* `N`：输入列数（int64）

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 CUDA `__global__` kernel，完成 per-tensor FP8 量化
