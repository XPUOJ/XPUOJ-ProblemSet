你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def per_tensor_quant_fp8(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    input,  # Tensor[bfloat16]
    output_q,  # Tensor[float8_e4m3fn]
    scale,  # Tensor[float32]
    is_static,  # int64
    M,  # int64
    N,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = per_tensor_quant_fp8(...)
    real_kernel(input, output_q, scale, is_static, M, N)
```

### 参数说明

* `input`：输入 tensor（bfloat16），shape $(M, N)$，行优先连续存储，只读
* `output_q`：输出 tensor（FP8 E4M3FN），shape $(M, N)$，需写入量化结果
* `scale`：输出 tensor（float32 标量），需写入计算的缩放因子
* `is_static`：int64 标量（0/1）
* `M`：输入行数
* `N`：输入列数

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 tilelang kernel，完成 per-tensor FP8 量化
