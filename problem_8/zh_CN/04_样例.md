## 输入

`x`（shape 为 $(2, 2)$，其中 $T=2, H=2$）：

```
[[1.0, 2.0],
 [0.0, -1.0]]
```

## 输出

`y`（shape 为 $(2, 2)$）：

```
[[0.7311, 1.7616],
 [0.0, -0.2689]]
```

## 计算过程说明

SiLU 函数对输入矩阵的每个元素独立进行计算：

* $x_{0,0} = 1.0$：$\sigma(1.0) \approx 0.7311$，$y_{0,0} = 1.0 \times 0.7311 \approx 0.7311$
* $x_{0,1} = 2.0$：$\sigma(2.0) \approx 0.8808$，$y_{0,1} = 2.0 \times 0.8808 \approx 1.7616$
* $x_{1,0} = 0.0$：$\sigma(0.0) = 0.5$，$y_{1,0} = 0.0 \times 0.5 = 0.0$
* $x_{1,1} = -1.0$：$\sigma(-1.0) \approx 0.2689$，$y_{1,1} = -1.0 \times 0.2689 \approx -0.2689$
