你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import tilelang
import tilelang.language as T
from tilelang import jit

real_kernel = None

@jit
def silu(*args):
    @T.prim_func
    def kernel(*args):
        ...
    return kernel

def run_kernel(
    x,  # Tensor[bfloat16]
    y,  # Tensor[bfloat16]
    T,  # int64
    H,  # int64
):
    global real_kernel
    if real_kernel is None:
        real_kernel = silu(...)
    real_kernel(x, y, T, H)
```

### 参数说明

* `x`：输入 tensor（bfloat16），shape $(T, H)$，行优先连续存储，只读
* `y`：输出 tensor（bfloat16），shape $(T, H)$，行优先连续存储，需写入结果
* `T`：第一维大小
* `H`：第二维大小

`run_kernel` 内部需要完成以下工作：

* 根据数据规模计算合适的 grid 和 block 配置
* launch 你实现的 tilelang kernel，完成 SiLU 计算
