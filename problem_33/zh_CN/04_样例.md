## 输入

```text
N = 1
C_in = 1
C_out = 1
H = W = 3
R = S = 3
stride = 1
padding = 0
```

此时：

```text
H_out = (3 + 0 - 3) / 1 + 1 = 1
W_out = (3 + 0 - 3) / 1 + 1 = 1
```

所以：

```text
temp.shape = (1, 1, 1, 1)
output.shape = (1, 1, 1, 1)
```

卷积先生成一个 `(1, 1, 1, 1)` 的中间结果，然后再与同形状的 `broadcast` 逐元素相加，得到最终输出。
