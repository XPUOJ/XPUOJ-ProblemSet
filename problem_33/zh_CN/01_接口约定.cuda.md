你必须在提交的 CUDA 源码中提供如下 **C 符号**，函数名、参数类型、顺序必须完全一致，并使用 `extern "C"` 防止 name mangling：

```cpp
#include <stdint.h>
#include <cuda_bf16.h>

extern "C" void run_kernel(
    const __nv_bfloat16* x,
    const __nv_bfloat16* weight,
    const __nv_bfloat16* bias,
    const __nv_bfloat16* broadcast,
    __nv_bfloat16* output,
    int64_t N,
    int64_t C_in,
    int64_t C_out,
    int64_t H,
    int64_t W,
    int64_t R,
    int64_t S,
    int64_t stride,
    int64_t padding
);
```

### 参数说明

* `x`：输入指针，shape `(N, C_in, H, W)`，类型为 bfloat16
* `weight`：输入指针，shape `(C_out, C_in, R, S)`，类型为 bfloat16
* `bias`：输入指针，shape `(C_out,)`，类型为 bfloat16
* `broadcast`：输入指针，shape `(N, C_out, H_out, W_out)`，类型为 bfloat16
* `output`：输出指针，shape `(N, C_out, H_out, W_out)`，类型为 bfloat16
* `N`：batch size
* `C_in`：输入通道数
* `C_out`：输出通道数
* `H`、`W`：输入高度、宽度
* `R`、`S`：卷积核高度、宽度
* `stride`：步长
* `padding`：零填充大小

`run_kernel` 内部需要完成以下工作：

* 根据输入尺寸计算输出形状
* 自行计算合适的 launch 配置
* launch 你实现的 CUDA kernel，完成二维卷积前向传播与 broadcast 加法
