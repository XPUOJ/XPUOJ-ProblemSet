你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序、类型必须完全一致：

```python
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(
    x,  # Tensor[bfloat16]
    weight,  # Tensor[bfloat16]
    bias,  # Tensor[bfloat16]
    broadcast,  # Tensor[bfloat16]
    output,  # Tensor[bfloat16]
    N,  # int64
    C_in,  # int64
    C_out,  # int64
    H,  # int64
    W,  # int64
    R,  # int64
    S,  # int64
    stride,  # int64
    padding,  # int64
):
    ...
```

### 参数说明

* `x`：输入 tensor，shape `(N, C_in, H, W)`，类型为 `bfloat16`
* `weight`：输入 tensor，shape `(C_out, C_in, R, S)`，类型为 `bfloat16`
* `bias`：输入 tensor，shape `(C_out,)`，类型为 `bfloat16`
* `broadcast`：输入 tensor，shape `(N, C_out, H_out, W_out)`，类型为 `bfloat16`
* `output`：输出 tensor，shape `(N, C_out, H_out, W_out)`，类型为 `bfloat16`
* 其余参数含义与 CUDA 接口一致

`run_kernel` 内部需要完成以下工作：

* 根据输入尺寸计算输出形状
* 调用你实现的 Triton kernel，或使用等价的 GPU Tensor 算子，完成二维卷积前向传播与 broadcast 加法
