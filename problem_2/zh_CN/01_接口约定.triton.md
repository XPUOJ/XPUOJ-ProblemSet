你必须在提交的 Python 代码中提供 `run_kernel` 函数，函数名、参数顺序必须完全一致：

```python
import torch
import triton
import triton.language as tl

@triton.jit
def your_kernel(...):
    ...

def run_kernel(A, B, C, M, N, K):
    ...
```

### 参数说明

* `A`：输入 tensor（bfloat16），shape $(M, K)$，行优先连续存储，只读
* `B`：输入 tensor（bfloat16），shape $(N, K)$，行优先连续存储，只读
* `C`：输出 tensor（bfloat16），shape $(M, N)$，行优先连续存储，需写入计算结果
* `M`：A 的行数，也是 C 的行数
* `N`：B 的行数，也是 C 的列数
* `K`：A 的列数，也是 B 的列数

`run_kernel` 内部需要完成以下工作：

* 根据矩阵尺寸计算合适的 grid 和 block 配置
* launch 你实现的 triton kernel，完成 $C = A \times B^T$ 的计算
